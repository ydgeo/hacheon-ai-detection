# 🌊 하천 점용 AI 추정 시스템

QGIS와 Google Gemini AI를 활용하여 항공사진에서 하천 점용 구역을 자동으로 탐지하는 시스템입니다.

---

## ✨ 주요 기능

- 🤖 **Google Gemini 2.5 Flash** AI로 항공사진 자동 판독
- 🖤 **암막(블라인드) 처리** — 폴리곤 경계 외부를 검정으로 가려 오탐 최소화
- ⚡ **병렬 처리** — 최대 8개 구역 동시 분석으로 빠른 처리
- 🗺️ **QGIS 레이어 자동 생성** — 의심 구역을 지도에 바로 표시
- 🔍 **복합 감지** — 한 구역에서 여러 위반 항목 동시 탐지
- 🖼️ **Few-shot 학습** — 실제 오탐 사례 이미지로 정확도 향상

### 탐지 항목
| 항목 | 설명 |
|------|------|
| 비닐하우스 | 흰색/반투명 아치형 구조물 |
| 가설건축물 | 컨테이너, 패널 조립 건물 |
| 경작지 | 규칙적 이랑(두둑) 패턴 |
| 자재적치 | 골재 더미, 폐기물 야적 |

## 🛠️ 설치 방법

### 1. QGIS 설치
[qgis.org](https://qgis.org) 에서 QGIS 3.x 다운로드 및 설치

### 2. OSGeo4W Shell에서 패키지 설치
**OSGeo4W Shell을 반드시 관리자 권한으로 실행 후** 아래 명령어 입력

```bash
python -m pip install --upgrade pip
pip install google-generativeai
pip install Pillow
```

설치 확인
```bash
python -c "import google.generativeai as genai; print('genai OK')"
python -c "from PIL import Image; print('Pillow OK')"
```

### 3. Google AI Studio API 키 발급
1. [aistudio.google.com](https://aistudio.google.com) 접속
2. **Get API key** → **Create API key** 클릭
3. 발급된 키 복사

## 📊 결과 확인

실행 완료 후 `AI_단속_의심구역` 레이어가 자동 생성됩니다.

### QGIS 속성 필터 활용
| 원하는 결과 | 필터 조건 |
|------------|----------|
| 경작지만 보기 | `"경작지" = 1` |
| 비닐하우스만 | `"비닐하우스" = 1` |
| 복합 위반 구역 | `"count" >= 2` |
| 가설건축물만 | `"가설건축물" = 1` |

### 카테고리별 색상 분류
레이어 더블클릭 → **심볼** → **단계 구분** → `count` 필드 선택

### 저장된 속성 필드
| 필드명 | 내용 |
|--------|------|
| detection | 감지된 항목 전체 ("비닐하우스, 경작지") |
| count | 위반 항목 수 |
| zone_id | 구역 번호 |
| 비닐하우스 | 감지 여부 (0 또는 1) |
| 가설건축물 | 감지 여부 (0 또는 1) |
| 경작지 | 감지 여부 (0 또는 1) |
| 자재적치 | 감지 여부 (0 또는 1) |

## 🖼️ 오탐 예시 이미지 추가 방법

오탐(잘못된 탐지)이 발생하면 해당 이미지를 예시로 추가해서 정확도를 높일 수 있습니다.

1. 오탐된 구역의 캡처 이미지를 `오탐예시` 폴더에 복사
2. 코드의 `FEWSHOT_EXAMPLES` 리스트에 추가

```python
{"filename": "새오탐예시.png", "answer": "정상", "reason": "오탐 이유 설명"}
```

## ❓ 자주 묻는 문제

**Q. `Invalid operation: The response.text quick access` 에러**
→ Gemini 2.5는 thinking 모드로 응답이 2개 파트로 옴. 최신 코드에서 이미 수정됨.

**Q. QGIS가 두 번째 실행 시 다운됨**
→ `mask_layer = None` 처리 없이 레이어 삭제 시 발생. 최신 코드에서 수정됨.

**Q. 건천(물 없는 하천) 바닥을 자재적치로 오탐**
→ `오탐예시` 폴더에 해당 이미지 추가 후 `FEWSHOT_EXAMPLES`에 등록.

**Q. 경계 밖 건물을 내부로 오탐**
→ 암막 처리가 적용되어 있으므로 최신 버전 사용 권장.
